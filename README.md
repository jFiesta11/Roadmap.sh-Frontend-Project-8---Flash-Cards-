# Flash Cards React App

## Features:

- Uses React-DOM
- Features Component based structuring and functioning
- Keys can be used for quicker navigation
- Questions and Answers are JSON based data
- Progress Bar that is calculated via length of data and current data
- Previous, Flip, Next, and Shuffle controls

## Installation

1. Clone the repository

```bash
   git clone https://github.com/jFiesta11/Roadmap.sh-Frontend-Project-8---Flash-Cards-
   cd Roadmap.sh-Frontend-Project-8---Flash-Cards-
```

2. Install dependencies

```bash
   npm install
```

3. Run the development server

```bash
   npm run dev
```

4. Open your browser to the URL shown in the terminal (usually `http://localhost:5173`)
